# 🍱 FoodVision AI — Smart Food Calorie Estimator

An AI-powered Food Recognition & Nutrition Analysis web app built using TensorFlow, Flask, and MobileNetV2.

This project can:
- 🍕 Detect food items from images
- 🔥 Estimate calorie content
- 📊 Display nutrition analytics
- 💪 Show protein, carbs, and fats
- 🧠 Calculate BMI & diet recommendations
- 🌙 Modern dark glassmorphism UI
- 📈 Interactive charts & food history

# 🖥️ Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/PRODIGY_ML_05.git
```

## Move Into Folder

```bash
cd PRODIGY_ML_05
```

## Create Virtual Environment

```bash
python -m venv venv
```

## Activate Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

---

# 📦 Install Requirements

```bash
pip install -r requirements.txt
```

---

# ▶️ Run Project

```bash
python app.py
```

Open:

```bash
http://127.0.0.1:5000
```
